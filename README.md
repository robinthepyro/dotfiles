<h1>Robin's Dotfiles</h1>
<h3>Hello!</h3>
<p>These are (the important parts of) my personal dotfiles. Currently I use these on a handful of old laptops and desktops and this is just a fancy way for me to share these files when I inevitably decide to hope distros again.</p>

<p>The focus of this config is to be lightweight and minimalist yet not lacking important features. This absolutely is still a work in progrees and these files will make no sense to anyone but me, though you are welcome to try get them to work on your system.</p>

<p>The core packages that this config is built around are as follows:</p>
<ul>
  <li>Hyprland - the best looking window manager out there, and it is very very configurable.</li>
  <li>waybar - specifically I am using waybar-cava (available in the AUR). Displays system info, a clock, and a simple audio visualiser thanks to Cava</li>
  <li>tofi - alternative app launcher. Designed to be very very fast, however currently I believe I cannot get it to launch terminal apps</li>
  <li>Kitty - it's just better than every other terminal...</li>
  <li>zsh - because it's better than bash</li>
  <li>helix - a really nice modal text editor (i live in this thing). Don't be like me tho, just use neovim</li>
  <li>dunst - a notification daemon, and imo it's better than mako</li>
  <li>yazi - a really nice TUI file manager with vim like bindings</li>
  <li>hyprpaper / hyprswitcher / hyprlock / hypridle - they just work too well to not use.</li>
</ul>
